<html> - Корневой тег

<head> - Добавление META инф. которая описывает страницу (ссылки на таб. стилей, на JS faills...)

<body> - В этом теге# добавляются все остальные теги которые используються для ВебСтраницы)

<div> - (разделение) возможность добавлять различные табл. стия. 

<a> - добавление гипер ссылок ( как на самом сайте так т на другие, внешние сайты)

<img> - Возможность вставлять на веб страницу изображения и указывать их место нахождения (файлов).

<h1> - Добавить заголовок в теле страницы (h1/h2/h3/....)

<p> - Добавление больших блоков текста из нескольких предложений.

<meta> - Используется в нутри тега# <head>. Добавление META инф. которая описывает страницу. 
Имено ето описание использует GOOGLE для поиска.

<style> - стилии CSS ( рекомендуется #<link> )
